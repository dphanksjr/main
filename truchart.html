Directions Form

Project Description

Program written for shuttle buses that transport residents to their homes. The web application connects to a third party datasource (MySQL)to pull up the Resident's address which automatically populates the Ending Address Text box. Pressing the Get Directions button uses Bing's API to display the directions in HTML format. The user can then save the directions to an SQL database and pull up a report for the driver's use.

Project Details

  • C#/ASP.Net/HTML/CSS
  • MySQL and SQL Server 2008
  • Used Visual Studio, Crystal Reports, Dreamweaver and Photoshop
  • Directions Form CS file example
