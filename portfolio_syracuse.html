The Directions Form used the following tools:

  • C#/ASP.Net/HTML/CSS
  • MySQL and SQL Server 2008
  • Used Visual Studio, Crystal Reports, Dreamweaver and Photoshop


Description:

Program written for shuttle buses that transport residents to their homes. The web application connects to a third party datasource (MySQL)to pull up the Resident's address which automatically populates the Ending Address Text box. Pressing the Get Directions button uses Bing's API to display the directions in HTML format. The user can then save the directions to an SQL database and pull up a report for the driver's use.

Loretto Reports used the following tools:

  • C#/ASP.Net/CSS
  • MySQL and SQL Server 2008
  • Used Visual Studio, Crystal Reports, Photoshop


Description:

Loretto Reports is a list of commonly used reports and labels used at individual sites. The tree view listing uses Active Directory permissions so individuals only see the site they work at, along with the individual reports they have permission for. The use of Active Directory security groups allows for an individual to have access as many as all reports and as little as one report. The user clicks on the link for the report they want and it grabs the data for the location selected.
